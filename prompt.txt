Создадим небольшой, но рабочий проект. 
1. Необходима база данных в формате json с полями id, name, email, phome,status
2. ФУнкции js
2.1 getAllContacts() - возвращает все контакты
2.2 addContact(contactData) - добавляем новый контакт
2.3 findActiveContact - возвращает только контакты со статосом активноый, то есть мы считываем активный статус из ячейки status в фа йле json и показываем активные контакты. Активным контактом будет считать контакт с 1, не активный контакт будет считать с 0

3. HTML страница (js можно прям туда внедрить без дополнительного встраивания). Должен быть красивый интерфейс, можно использовать taildiwnd css, цвета должны быть надоподие как в приложении (сайте) vk.com. 
На странице можно будет вывести список активных контактов, возможность добавления контактов, удаление контактов, изменение. 
не забудь про кнопки, добавь так же сортировку по id, name,email,phone, status - соотвественно они должны быть на русском языке

Функции добавления контакта - мы должны иметь возможность добавить контакт через консоль в браузере (так же добавь пример в README.md)
Ко всем функциям мы так же должны иметь возможность обращаться из консоли. 

Добавь 3 тестовых данных контактов, для проверки и оценки работоспособности. 

И да, использовать обычный html, js без npm
